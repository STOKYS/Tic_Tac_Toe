# Tic_Tac_Toe 
## David Stoček

### What is it?

This repo is used to store simple game of Tic Tac Toe written in **JavaScript**.

#### Features:

- Resizable grid
- Changeable points to win
- Activating helping grid
- **GAMER MODE**

#### Upcoming Features

- Working Start and End
- Working Scoreboard
